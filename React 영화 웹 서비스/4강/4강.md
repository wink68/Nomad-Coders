## 4강 : Props
### 1. shortcut   
* 일반적인 props 방법   
```
function Btn(props) {
  return (
    <button>{props.text}</button>
  );
}
```

<br>

* shortcut (지름길)   
```
function Btn({ text }) {
  return (
    <button>{text}</button>
  );
}
```

<br>
